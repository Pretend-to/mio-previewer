使用方法

在 Vue 组件中使用 cursorPlugin 配置光标样式：

import { MdRenderer } from 'mio-previewer' import { cursorPlugin } from 'mio-previewer/plugins/custom' // 1. 默认方形光标 const plugins = [ { plugin: cursorPlugin } ] // 2. 竖线光标 const plugins = [ { plugin: cursorPlugin, options: { shape: 'line', color: '#0066ff' } } ] // 3. 圆形黑色光标 ✨ 新增 const plugins = [ { plugin: cursorPlugin, options: { shape: 'circle', color: '#000' } } ] // 4. 圆形绿色光标 const plugins = [ { plugin: cursorPlugin, options: { shape: 'circle', color: '#28a745' } } ]

类型定义

interface CursorPluginOptions { /** * 光标形状 * - 'square': 方形光标（默认） * - 'line': 竖线光标 * - 'circle': 圆形光标 ✨ 新增 */ shape?: 'square' | 'line' | 'circle'; /** * 光标颜色（CSS 颜色值） * 默认: '#333' */ color?: string; }

样式规格

  • 方形光标： 8px × 1.2em
  • 竖线光标： 2px × 1.2em
  • 圆形光标： 8px × 8px，圆角 50%
  • 动画效果： 透明度从 1 → 0.2 → 1 平滑过渡（1.2秒循环）

🎨 推荐搭配

  • 黑色圆形光标：适合浅色背景，提供经典优雅的视觉效果
  • 蓝色圆形光标：适合链接和交互元素，提供现代科技感
  • 绿色圆形光标：适合终端/控制台场景，呈现技术氛围
